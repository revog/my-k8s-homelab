# Flux

## Create pre-templated Flux manifests for Cluster bootstrapping

```bash
# Overwrite file with a clear header
echo "# This manifest was generated by automation. DO NOT EDIT." > kubernetes/bootstrap/flux/flux.yaml

# Export fully rendered Flux manifests
flux install --namespace infra-flux --export >> kubernetes/bootstrap/flux/components.yaml

# Create mono yaml with all ressources created during bootstrap
kustomize build > kubernetes/bootstrap/flux/flux.yaml
```

⚠️ This pre-rendered file is only used during cluster bootstrapping process. After successfull deployment of Flux, application will be maintained through CICD mechanisms.
