# Cilium

## Create pre-templated Cilium manifests for Cluster bootstrapping

```bash
# Overwrite file with a clear header
echo "# This manifest was generated by automation. DO NOT EDIT." > talos/bootstrap/cilium/cilium.yaml

# Render Cilium deployment with Helm
helmfile -f talos/bootstrap/cilium/helmfile.yaml template >> talos/bootstrap/cilium/cilium.yaml
```
⚠️ This pre-rendered file is only used during cluster bootstrapping process. After ArgoCD is deployed successfully, application will be maintained through ArgoCD mechanisms.
