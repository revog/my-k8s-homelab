# ArgoCD

## Create pre-templated ArgoCD manifests for Cluster bootstrapping

```bash
# Overwrite file with a clear header
echo "# This manifest was generated by automation. DO NOT EDIT." > kubernetes/bootstrap/argocd/argocd.yaml

# Render Cilium with Helm
helmfile -f kubernetes/bootstrap/argocd/helmfile.yaml template >>  kubernetes/bootstrap/argocd/argocd.yaml
```

⚠️ This pre-rendered file is only used during cluster bootstrapping process. After ArgoCD is deployed successfully, application will be maintained through ArgoCD mechanisms.
